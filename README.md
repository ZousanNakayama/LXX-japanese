# LXX（七十人訳聖書）日本語訳プロジェクト

## 概要 
このリポジトリは、**ギリシア語七十人訳聖書（Septuagint, LXX）を日本語に翻訳し、無料で公開する**プロジェクトです。  
七十人訳聖書は、紀元前1世紀ごろからユダヤ人や初期キリスト教徒に広く用いられていた、旧約聖書の重要なギリシア語訳です。現在主流のマソラ本文とは異なる内容を多く含み、信仰的・学術的に非常に価値があります。

しかし、日本語訳はこれまで限られており、入手も困難でした。  
本プロジェクトでは、**世界初の英語訳である1808年のチャールズ・トムソン訳（パブリックドメイン）** をもとに、逐語訳を重視した日本語訳を進めています。

## 📖 Charles Thomson Bible とは
**Charles Thomson Bible（1808）** は、アメリカ合衆国建国の父の一人、チャールズ・トムソンがギリシア語七十人訳聖書（Septuagint, LXX）から直接英語に忠実に翻訳した歴史的に重要な聖書です。  
この翻訳聖書の主な特徴は以下の通りです。
- 4世紀のバチカン写本を反映した、17世紀の印刷本を底本とする
- Thomson は約 20 年をかけて原文を精査し、原語に忠実に、意訳を避けた翻訳を完成した  
- 1808 年版は **著作権が完全に消滅しており Public Domain**となっている
