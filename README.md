# A-Japanese-translation-of-the-Charles-Thomson-Bible
チャールズ・トムソン訳聖書の日本語訳

## Charles Thomson Bible について

Charles Thomson Bible（1808年）は、ギリシア語七十人訳聖書（Septuagint）を
英語へできるだけ忠実に翻訳することを目的として作られた聖書です。

当時の英語圏ではヘブライ語原典からの翻訳が主流で、
七十人訳に基づく英語訳はほとんど存在していませんでした。
Thomson は、初期教会で重視されていた七十人訳を英語で直接読めるようにするため、
約20年をかけて原文を精査し、過度な意訳を避けた翻訳を完成させました。

この 1808 年版は著作権が完全に切れており、現在は Public Domain として自由に利用できます。

## 日本語訳の方針

このリポジトリでは、Charles Thomson 訳（Public Domain）をもとに、
以下の方針で日本語訳を作成しています。

- 原文にできるだけ忠実  
- 逐語訳寄りで、省略しない  
- 意味を簡単にしすぎず、語感が透けて見える訳を目指す  
- 原文の雰囲気・構造・論理を日本語にそのまま映すことを重視
