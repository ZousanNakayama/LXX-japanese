# Charles Thomson Bible 日本語訳プロジェクト  
**トムソン訳聖書（Charles Thomson Bible, 1808）に忠実な日本語訳**

このリポジトリは、**Charles Thomson Bible（1808）** を日本語で読めるようにするための翻訳プロジェクトです。  
本翻訳は、ギリシア語七十人訳聖書（Septuagint, LXX）そのものではなく、  
**「トムソン訳（Thomson’s Translation）」に忠実であること** を最優先としています。

Charles Thomson Bible は、七十人訳聖書を英語へ「できるだけ忠実に」翻訳した歴史的資料であり、  
その英語訳の語感・構造・論理を日本語に移すことを目的としています。

---

## 📘 なぜ「トムソン訳」に忠実なのか（日本語ユーザー向けの背景）

### 🔹 七十人訳（LXX）そのものではなく「トムソン訳」を翻訳する理由
日本語で七十人訳聖書（LXX）を読む方法は限られていますが、  
**LXX の逐語的な英語訳であるトムソン訳（1808）は完全に Public Domain** であり、  
日本語へ翻訳する際の法的・実務的な障壁がありません。

さらに、トムソン訳は：

- 原文の語順・構造をできるだけ保持した英語訳  
- 過度な意訳を避けた、学術的価値の高い翻訳  
- LXX の特徴を英語で読み取れる貴重な資料  
- 1808 年版は著作権が完全に消滅しており自由に利用可能  

つまり、**「LXX の雰囲気を保った英語訳」→「日本語訳」** という形で、  
原文のニュアンスを損なわずに日本語へ移すことができます。

### 🔹 このプロジェクトの目的
- トムソン訳聖書（1808）を日本語で読めるようにする  
- 英語訳の語感・構造・論理をできるだけ忠実に再現する  
- LXX とヘブライ語原典の違いを日本語で比較できる資料を提供する  
- 聖書研究者・言語学習者・信仰者のための新しい選択肢を作る  

---

## 📝 日本語訳の方針（翻訳ポリシー）
- **LXX ではなく「トムソン訳」に忠実であることを最優先**  
- 原文英語の語感・構造をできるだけ保持  
- 逐語訳寄りで、省略しない  
- 意味を単純化しすぎず、英語の論理が透けて見える訳を目指す  
- トムソン訳の特徴（語順・接続詞・文構造）を可能な限り保持

## 📖 Charles Thomson Bible とは
**Charles Thomson Bible（1808）** は、ギリシア語七十人訳聖書（Septuagint, LXX）を  
英語へ忠実に翻訳した最初期の聖書です。

- 当時の英語圏ではヘブライ語原典からの翻訳が主流  
- 七十人訳に基づく英語訳はほとんど存在しなかった  
- Thomson は約 20 年をかけて原文を精査し、過度な意訳を避けた翻訳を完成  
- 1808 年版は **著作権が完全に消滅しており Public Domain**
